# SmartSpend

MERN-based personal expense tracker with:
- `backend` (Node.js + Express + MongoDB)
- `frontend` (React + Vite)

## Setup
Copy env templates before running:

```bash
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
```

## Run frontend
```bash
cd frontend
npm install
npm run dev
```

## Run backend
```bash
cd backend
npm install
npm run dev
```
